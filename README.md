•	Pasos para ejecutar el proyecto:
Una vez tengas abierto el proyecto en IntelliJ, das click a el botón play en la parte superior del programa, si esta opción no está disponible, puedes ingresar a la sección Gradle que se encuentra en
la barra derecha del programa y se lo identifica con un icono de un elefante. Una vez en esta sección das click en rest después en Tasks, después application y por último en bootRun. Se desplegará una ventana donde
se muestra que el proyecto está corriendo.
Para el siguiente paso tienes dos opciones:
1. Usar Postman
2. Usar un navegador

*En Postman usas un método GET y colocas localhost:8080/square o localhost:8080/aloooo, les das click en SEND y podrás observar el mensaje.
*En tu navegador escribes localhost:8080/square o localhost:8080/aloooo en la barra de direcciones das ENTER y podrás observar el mensaje.
